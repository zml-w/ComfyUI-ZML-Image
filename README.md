# ComfyUI-ZML-Image
这是一个帮助管理图像的节点插件。相信很多朋友在用ComfyUI读取tag时都遇到过这样的困扰：要在复杂的工作流中费力地寻找文本输入节点来复制tag；想保存tag和对应图像时，发现市面上没有既免费好用、又能方便保存图像和tag的软件。
为此我开发了这个节点，它能让你：
1、直接通过系统自带的文件管理器管理图像，无需购买软件授权。
2、将文本写入到图像里，需要特定节点来提取，在外面完全看不出来区别，让图片在文件夹里更干净简洁。
3、支持从随机文件夹中抽取图像和对应tag（抽卡抽到爽！）。
4、我还开发了一个将图片/GIF转为HTML文件的节点，在外面完全看不出来这个文件里是什么，必须下载后在本地打开才会显示图像/GIF里的内容，至于这个功能主要在什么时候使用...你知道的_
